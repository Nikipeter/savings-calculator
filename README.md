# 💰 Six Jar Savings Calculator

A simple and user-friendly web application that helps you divide your total savings using the **Six Jar Method** of money management.

---

## 📌 About the Project

This calculator allows users to enter their total savings amount and instantly see how their money should be distributed across six categories based on a proven financial strategy.

It also includes a **Time Difference Calculator** to calculate duration between two times in both **AM/PM** and **24-hour formats**.

---

## 🚀 Features

### 💰 Savings Calculator (Six Jar Method)
- Enter your total amount
- Automatically divides into:
  - **Utilities (50%)**
  - **Short-term Savings (20%)**
  - **Long-term Savings (15%)**
  - **Charity (5%)**
  - **Financial Freedom (5%)**
  - **Play (5%)**
- Clean and easy-to-read output format

---

### ⏱ Time Difference Calculator
- Calculate difference between two times
- Supports:
  - **AM/PM format**
  - **24-hour format**
- Handles overnight calculations  
  *(Example: 22:00 → 06:00 = 8 hours)*
